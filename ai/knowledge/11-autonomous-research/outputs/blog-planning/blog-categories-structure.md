## Category Page SEO Elements

### Category Page Template

Each category page should include:

1. **H1 Title**: "[Category Name] | Foamology Insulation Blog"
2. **Meta Description**: 155 characters describing category content
3. **Category Description**: 150-200 word introduction
4. **Featured Posts**: 3 top posts in category
5. **Full Post Grid**: All posts with pagination (12 per page)
6. **Related Categories**: Links to 2-3 related categories
7. **CTA Section**: Appropriate call-to-action for funnel stage

### Example Category Page Structure

```
/blog/category/spray-foam-guides/

H1: Spray Foam Insulation Guides for Alaska Homeowners

Meta Description: Comprehensive spray foam insulation guides covering types, applications, costs, and installation. Expert information for Alaska homeowners from Foamology.

[Category Description - 150 words]

## Featured Guides
[3 featured posts]

## All Spray Foam Guides
[Post grid with filtering by tags]

## Related Categories
- Insulation Education
- Comparisons & Reviews
- Energy Savings

[CTA: Get Your Free Quote]
```

## Internal Linking Framework

### Category Hub Structure

Each category serves as a hub for internal linking:

```
[Category Hub Page]
       ↓
   [Featured Posts] ←→ [Related Posts within category]
       ↓
   [Pillar Content] ←→ [Supporting Content]
       ↓
   [Service Pages] ←→ [Contact/Quote Pages]
```

### Cross-Category Linking Rules

1. **Educational → Comparison**: Link to comparison when types are mentioned
2. **Comparison → Buyer Resources**: Link to buyer guides from conclusions
3. **FAQ → Educational**: Link to detailed articles for deeper answers
4. **Case Studies → Service Pages**: Link to relevant service offerings
5. **Energy Savings → ROI Calculator**: Link to interactive tools
6. **Alaska Solutions → Local Pages**: Link to service area pages

### Anchor Text Strategy

**For Category Pages:**
- Use category name as anchor text
- Example: "Browse our [spray foam guides] for more information"

**For Posts Within Category:**
- Use post title or descriptive phrase
- Example: "Learn more in our [complete buyer's guide to spray foam]"

**For Cross-Category Links:**
- Use benefit-focused anchor text
- Example: "See how much you could save in our [energy savings section]"
